# InputMediaSettings

Displays the result of the input Media settings.

## Example Usage

```typescript
import { InputMediaSettings } from "@fastpix/fastpix-node/models";

let value: InputMediaSettings = {
  metadata: {
    "livestream_name": "fastpix_livestream",
  },
};
```
