# MediaMaxResolution

The maximum resolution specified by the user for the media.

## Example Usage

```typescript
import { MediaMaxResolution } from "@fastpix/fastpix-node/models";

let value: MediaMaxResolution = "1080p";
```

## Values

```typescript
"2160p" | "1440p" | "1080p" | "720p" | "480p" | "360p"
```
