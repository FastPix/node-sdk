# SubtitleTrack

A media consists of different media tracks, like video, audio, and subtitle, all combined.

## Example Usage

```typescript
import { SubtitleTrack } from "@fastpix/fastpix-node/models";

let value: SubtitleTrack = {
  id: "9oa85f64-5717-4562-b3fc-2c963f66afa6",
  type: "subtitle",
  status: "available",
  languageName: "english",
  languageCode: "en",
};
```
