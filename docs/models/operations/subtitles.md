# Subtitles

Generates subtitle files for audio/video files.


## Example Usage

```typescript
import { Subtitles } from "@fastpix/fastpix-node/models/operations";

let value: Subtitles = {
  languageName: "english",
  metadata: {
    "key1": "value1",
  },
  languageCode: "en",
};
```
