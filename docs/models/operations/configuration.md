# Configuration

Represents configuration details for the media.

## Example Usage

```typescript
import { Configuration } from "@fastpix/fastpix-node/models/operations";

let value: Configuration = {
  url: "https://static.fastpix.io/fp-sample-video.mp4",
};
```
