# MaxResolution

Determines the highest quality resolution available.


## Example Usage

```typescript
import { MaxResolution } from "@fastpix/fastpix-node/models/operations";

let value: MaxResolution = "1080p";
```

## Values

```typescript
"2160p" | "1440p" | "1080p" | "720p" | "480p" | "360p"
```
